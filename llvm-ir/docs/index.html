Compilation Model

LLVM IR assembly code can be compiled to native assembly code,
which then can be assembled and linked into a native executable.

Alternatively, LLVM IR assembly code can be assembled into bitcode,
which can be linked together into a portable executable.
Such portable executable can be interpreted or further processed into a native executable.

Note that, linking LLVM IR bitcode files into portable executable,
enables the possibility of performing link-time optimizations.
